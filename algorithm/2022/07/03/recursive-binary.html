재귀함수를 이용한 이진수 출력 문제

Jul 3, 2022 • 1 min read

algorithm

10진수 N이 입력되면 2진수로 변환하여 출력하는 프로그램을 작성하시오. 단 재귀함수를 이용해서 출력해야 합니다.

  • 입력 설명

    첫 번째 줄에 10진수 N(1≤ N ≤ 1000)이 주어집니다.

  • 출력 설명

    첫번째 줄에 이진수를 출력하세요

  • 입력 예제 1

    11

  • 출력 예제 1

    1011

def DFS(x):
	if x == 0 :
		return

	else :
		print(x%2, end=' ') # x를 2로 나눈 나머지를 출력하고
		DFS(x//2) # 2를 나눈 몫을 다음 재귀로 넘겨준다

if x = 16

DFS(16)
16 % 2 = 0 ---> 0
16 // 2 = 8

DFS(8)
8 % 2 = 0 ---> 0 0
8 // 2 = 4

DFS(4)
4 % 2 = 0 ---> 0 0 0
4 // 2 = 2

DFS(2)
2 % 2 = 0 ---> 0 0 0 0
2 // 2 = 1

DFS(1)
1 % 2 = 1 ---> ** 0 0 0 0 1 **
1 // 2 = 0

DFS(0)
